## About Me

I'm a **Full-Stack Developer** from Cavite, Philippines, passionate about building things that matter — from real-time collaborative tools and enterprise SaaS platforms to mobile-first offline apps and retro games in pure x86 Assembly.

My projects span **disaster response mapping**, **inventory management**, **e-commerce**, **content management**, **crowd-tracking for commuters**, **community food sharing**, and more. I approach every project with a focus on clean architecture, performance, and user experience.

- 🎓 **BSIT** · St. Dominic College of Asia — Dean's Lister
- 💼 **Open to work** — freelance and full-time opportunities
- 🌐 **Tech range** — web, mobile, desktop, and low-level programming
- ⚡ **Fun fact** — I've written an entire game in pure x86 Assembly
